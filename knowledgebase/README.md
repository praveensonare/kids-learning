# Knowledge Base Files

This directory contains curriculum knowledge base files for each class and subject.

## File Naming Convention

Files follow the pattern: `{year}_{subject}.txt`

Examples:
- `nursery_literacy.txt`
- `year1_english.txt`
- `year2_maths.txt`
- `year3_science.txt`

## Content Structure

Each file contains:
- Learning objectives aligned with Cambridge International Standards
- Key topics and concepts
- Sample activities and resources
- Age-appropriate content

## Usage

These files serve as reference material for:
- Curriculum planning
- Lesson content
- Assessment guidelines
- Parent information
