Nursery Literacy - Chapter 1: Alphabet Fun

TOPIC 1: Letter Recognition A-M

Introduction:
Learning the first half of the alphabet through multi-sensory activities.

Activities:
1. Sing alphabet songs focusing on A-M
2. Trace large letters in sand, paint, or with fingers
3. Match uppercase and lowercase letters
4. Find letters in the environment (signs, books, labels)
5. Create letter collages with objects starting with each letter

Key Vocabulary:
letter, alphabet, uppercase, lowercase, sound

Resources Needed:
- Alphabet flashcards
- Letter formation cards
- Environmental print

Assessment Checkpoint:
Can the child recognize and name letters A-M? Can they identify the initial sound?

---

TOPIC 2: Letter Recognition N-Z

Introduction:
Completing the alphabet with letters N-Z.

Activities:
1. Continue alphabet songs with N-Z
2. Letter hunts - finding specific letters
3. Sorting letters (straight lines vs curved)
4. Building letters with playdough
5. Letter sound games

Key Vocabulary:
complete alphabet, letter names, letter shapes, straight, curved

Resources Needed:
- Letter blocks
- Magnetic letters
- Letter stamps

Assessment Checkpoint:
Can recognize most letters of the alphabet? Beginning to associate sounds?

---

TOPIC 3: Initial Sounds

Introduction:
Understanding that words start with different sounds.

Activities:
1. I-Spy games with initial sounds
2. Sound sorting activities
3. Picture cards - matching initial sounds
4. Rhyme time - finding words that sound similar
5. Alliteration activities (e.g., "Silly Sally")

Key Vocabulary:
sound, begins with, starts with, rhyme, same sound

Resources Needed:
- Picture cards
- Sound boxes
- Rhyming books

Assessment Checkpoint:
Can identify initial sounds in familiar words?

---

TOPIC 4: Print Awareness

Introduction:
Understanding how books and print work.

Activities:
1. Book handling - showing correct orientation
2. Following text with finger during shared reading
3. Understanding that print carries meaning
4. Recognizing own name in print
5. Environmental print walks

Key Vocabulary:
book, story, front, back, page, words, read

Resources Needed:
- Big books
- Name cards
- Labels around classroom

Assessment Checkpoint:
Does child show understanding of how books work? Recognizes own name?
