Year 3 Maths - Cambridge Primary Curriculum

Learning Objectives:
- Count, read, write numbers to 1000
- Add and subtract 3-digit numbers
- Know 3, 4, 8 times tables
- Understand tenths and fractions
- Measure perimeter
- Tell time using 12-hour and 24-hour clocks
- Interpret bar charts and tables

Topics:

1, Place Value to 1000, Learn about place value to 1000
2, Addition and Subtraction, Learn about addition and subtraction
3, Multiplication and Division, Learn about multiplication and division
4, Fractions, Learn about fractions
5, Measurement, Learn about measurement
6, Geometry, Learn about geometry
7, Time, Learn about time
8, Statistics, Learn about statistics
