Year 4 English - Cambridge Primary Standards

Learning Objectives:


Topics:

4, Spelling: Prefixes (in-, im-, il-, ir-, sub-, inter-), suffixes (-ation, -ly, -ous), homophones (accept/except, whose/who's), apostrophes for possession/contraction, Learn about spelling: prefixes (in-, im-, il-, ir-, sub-, inter-), suffixes (-ation, -ly, -ous), homophones (accept/except, whose/who's), apostrophes for possession/contraction
5, Poetry Analysis: Figurative language, verse structure, rhythm and rhyme, performance, write own poems using techniques learned, Learn about poetry analysis: figurative language, verse structure, rhythm and rhyme, performance, write own poems using techniques learned
