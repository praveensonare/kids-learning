Nursery Numeracy - Chapter 4: Patterns and Positions

TOPIC 1: Simple Patterns

Introduction:
Recognizing and creating simple repeating patterns.

Activities:
1. Copying AB patterns (red, blue, red, blue)
2. Making patterns with objects, colors, shapes
3. Finding patterns in environment (floor tiles, fences)
4. Creating bead patterns and block patterns
5. Clapping and stamping rhythm patterns

Key Vocabulary:
pattern, repeat, next, same, different, copy, continue

Resources Needed:
- Pattern blocks
- Beads and laces
- Colored objects
- Musical instruments

Assessment Checkpoint:
Can copy a simple AB pattern? Recognizes when pattern repeats? Can say what comes next?

---

TOPIC 2: Repeating Patterns

Introduction:
Extending and creating patterns with different elements.

Activities:
1. Continuing started patterns (what comes next?)
2. Making ABB and AAB patterns
3. Fixing broken patterns (find the mistake)
4. Creating patterns with movement and sound
5. Pattern painting and printing

Key Vocabulary:
continue, extend, next one, repeating, sequence, mistake, fix

Resources Needed:
- Pattern cards
- Building materials
- Art supplies
- Sound makers

Assessment Checkpoint:
Can continue a pattern? Creates own simple patterns? Spots pattern errors?

---

TOPIC 3: Positional Language

Introduction:
Understanding and using position words.

Activities:
1. Following position instructions (put teddy on the chair)
2. Describing where things are (the ball is under the table)
3. Treasure hunts using position clues
4. Building and describing constructions
5. Obstacle courses (go over, under, through)

Key Vocabulary:
on, under, in, next to, behind, in front, between, beside, top, bottom

Resources Needed:
- Small world toys
- Large apparatus
- Position picture cards
- Everyday objects

Assessment Checkpoint:
Follows simple position instructions? Uses some position words? Understands on/under/in?

---

TOPIC 4: Size Ordering

Introduction:
Comparing and ordering by size.

Activities:
1. Sorting big and small objects
2. Ordering 3 bears/items by size
3. Finding biggest and smallest
4. Comparing height and length
5. Using size language in play

Key Vocabulary:
big, small, bigger, smaller, biggest, smallest, tall, short, long, tiny, huge

Resources Needed:
- Graduated sets (nesting toys, stacking cups)
- Size sorting cards
- Measuring tools
- Comparison objects

Assessment Checkpoint:
Can compare two objects (bigger/smaller)? Orders 3 items by size? Uses size vocabulary?
