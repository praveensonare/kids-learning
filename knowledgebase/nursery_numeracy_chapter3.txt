Nursery Numeracy - Chapter 3: Shapes and Space

TOPIC 1: Basic 2D Shapes

Introduction:
Exploring flat shapes in the environment and through play.

Activities:
1. Identifying circles, squares, triangles, rectangles
2. Shape hunts - finding shapes in classroom and outdoors
3. Sorting objects by shape
4. Making shape pictures and patterns
5. Playing with shape puzzles and sorters

Key Vocabulary:
circle, square, triangle, rectangle, shape, round, corners, sides, flat

Resources Needed:
- Shape blocks
- Shape sorters
- Shape pictures
- Play dough for making shapes

Assessment Checkpoint:
Can name 2-3 basic shapes? Can find shapes in environment? Beginning to describe shapes?

---

TOPIC 2: Basic 3D Shapes

Introduction:
Exploring solid shapes through hands-on manipulation.

Activities:
1. Exploring sphere (ball), cube (box), cylinder (can)
2. Feeling and describing 3D shapes
3. Building with blocks and boxes
4. Rolling and stacking investigations
5. Finding 3D shapes in everyday objects

Key Vocabulary:
ball, box, can, solid, roll, stack, build, round, straight

Resources Needed:
- 3D shape collection
- Building blocks
- Everyday objects (tins, balls, boxes)
- Construction materials

Assessment Checkpoint:
Can identify ball and box? Explores properties (rolling, stacking)? Uses shape names?

---

TOPIC 3: Shape Properties

Introduction:
Beginning to notice features of shapes.

Activities:
1. Counting corners and sides on shapes
2. Sorting shapes - straight sides vs curved
3. Feeling shape edges - smooth or pointy
4. Comparing big and small versions of same shape
5. Making shapes with sticks and playdough

Key Vocabulary:
corner, side, edge, curved, straight, round, pointy, same shape, different size

Resources Needed:
- Shape sets
- Sticks and connectors
- Playdough
- Feely bags

Assessment Checkpoint:
Notices some shape features? Can sort by simple properties? Uses describing words?

---

TOPIC 4: Sorting Shapes

Introduction:
Grouping shapes by different attributes.

Activities:
1. Sorting by shape name (all circles together)
2. Sorting by color
3. Sorting by size (big shapes, little shapes)
4. Sorting by number of corners
5. Creating own sorting rules

Key Vocabulary:
sort, group, same, different, belongs, doesn't belong, set

Resources Needed:
- Mixed shape collections
- Sorting hoops
- Shape attribute cards
- Containers for sorting

Assessment Checkpoint:
Can sort by one attribute? Explains sorting choice? Shows flexibility in sorting?
