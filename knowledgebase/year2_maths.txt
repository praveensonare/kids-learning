Year 2 Maths - Cambridge Primary Curriculum

Learning Objectives:
- Count, read, write numbers to 100, understanding place value
- Add and subtract 2-digit numbers
- Know 2, 5, 10 times tables
- Use and apply fractions
- Measure using standard units
- Identify and describe properties of 2D and 3D shapes
- Tell time to 5 minutes
- Interpret simple graphs and data

Topics:

1, Place Value to 100, Learn about place value to 100
2, Addition and Subtraction (2-digit numbers), Learn about addition and subtraction (2-digit numbers)
3, Multiplication and Division, Learn about multiplication and division
4, Fractions, Learn about fractions
5, Measurement, Learn about measurement
6, Geometry - 2D and 3D Shapes, Learn about geometry - 2d and 3d shapes
7, Time, Learn about time
8, Statistics and Data, Learn about statistics and data
