Nursery Numeracy - Chapter 1: Numbers 1-5

TOPIC 1: Counting to 5

Introduction:
Learning to count objects reliably to 5.

Activities:
1. Counting fingers on one hand
2. Counting objects in groups 1-5
3. Singing number rhymes (5 little ducks, 5 currant buns)
4. Touch-counting activities
5. Number hunts - finding groups of 1, 2, 3, 4, 5

Key Vocabulary:
one, two, three, four, five, count, how many, number

Resources Needed:
- Counting objects (blocks, buttons, toys)
- Number rhyme props
- Number cards 1-5

Assessment Checkpoint:
Can count up to 5 objects accurately using one-to-one correspondence?

---

TOPIC 2: Number Recognition 1-5

Introduction:
Recognizing and matching numerals 1-5.

Activities:
1. Number matching games
2. Finding numbers in environment
3. Number formation practice (large scale)
4. Matching numeral to quantity
5. Number bingo with numbers 1-5

Key Vocabulary:
number, numeral, match, same, different

Resources Needed:
- Number cards
- Magnetic numbers
- Number stamps
- Playdough

Assessment Checkpoint:
Can recognize numerals 1-5? Beginning to write them?

---

TOPIC 3: One-to-One Correspondence

Introduction:
Understanding that each object gets one count.

Activities:
1. Setting table - one plate per person
2. Sharing objects - one each
3. Counting with finger pointing
4. Matching games
5. Pairing activities (socks, gloves)

Key Vocabulary:
one each, same number, matching, pairs

Resources Needed:
- Everyday objects
- Counting collections
- Matching cards

Assessment Checkpoint:
Shows understanding of one-to-one correspondence when counting?

---

TOPIC 4: Comparing Quantities to 5

Introduction:
Using language to compare amounts.

Activities:
1. More and less games
2. Comparing towers of blocks
3. Which has more? activities
4. Sorting by size
5. Making equal groups

Key Vocabulary:
more, less, fewer, same, equal, bigger group, smaller group

Resources Needed:
- Counting collections
- Balance scales
- Comparison cards

Assessment Checkpoint:
Can use language of more/less/same to compare groups to 5?
